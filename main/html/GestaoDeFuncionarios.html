Gestão de Funcionários

Gerenciamento completo de funcionários, cargos e departamentos

Principais Recursos

  • Cadastro completo de funcionários com histórico profissional
  • Gerenciamento de departamentos e hierarquia organizacional
  • Controle de ponto e monitoramento de produtividade
  • Relatórios detalhados sobre desempenho e recursos humanos

Sobre o Projeto

Este projeto foi desenvolvido para facilitar a gestão de recursos humanos em empresas de todos os portes, oferecendo ferramentas completas para o controle de funcionários, departamentos e cargos.

Utilizando tecnologias modernas como Java e Spring Boot, o sistema de Gestão de Funcionários oferece uma API robusta e escalável com documentação completa via Swagger.

Tecnologias Utilizadas

Java
Spring Boot
Spring Data JPA + MySQL
Maven
Swagger
Postman
